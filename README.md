# gid-super-item-creator

Check the app [here](https://gid-super-item-creator.herokuapp.com/).

The app:
* Uses **LocalStorage** to store `data`
* Stores **images** as `base24 data`

Possible improvement:
* A proper data base
* Routing (with react-router)
